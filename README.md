# 安装依赖和开发者工具配置
 1. 在miniprogram目录运行：
 ``` npm install  ```
2. 在微信开发者编辑器中选择 "工具/构建npm"

Tips: 其他配置项目请查看腾讯官方配置文档（https://developers.weixin.qq.com/miniprogram/dev/devtools/npm.html）
